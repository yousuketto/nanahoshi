# Nanahoshi

Nanahoshi is web ui for gem dependencies graph.

## Installation

Add this line to your application's Gemfile:

```ruby
group :development do
  gem 'nanahoshi'
end
```

And then execute:

    $ bundle

## Usage
access to ```http://localhost:3000/rails/info/gems```
